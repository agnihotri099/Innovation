# Innovation

Deploy site link : https://agnihotri099-innovation.netlify.app/login

## Local Development

To run the project locally:

1. Clone the repository.
2. Install dependencies using `npm install`.
3. Start the development server using `npm start`.

## Dependencies

- react-router-dom
- react-redux
- redux
